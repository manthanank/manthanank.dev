# Manthanank.dev

Personal portfolio and blog built with Astro.

## Features

- Portfolio pages (home, about, projects, contact)
- Blog powered by Astro content collections
- RSS feed and sitemap
- Responsive design

## Tech Stack

- Astro
- TypeScript
- CSS

## Getting Started

Install dependencies:

```bash
npm install
```

Run the dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Project Structure

```tree
public/
  manifest.json
  robots.txt
src/
  components/
  content/
    blog/
  layouts/
  pages/
  styles/
```

## Content

Blog posts live in `src/content/blog` as Markdown files. Update frontmatter and content there to add or edit posts.
